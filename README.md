# Trabalho de Teoria da Informação

## Como executar?

Primeiro instale o streamlit

```bash
pip install streamlit run ./src/app.py
```

Em seguida execute o comando abaixo para executar o ambiente na máquina local:

```bash
streamlit run ./src/main.py
```
